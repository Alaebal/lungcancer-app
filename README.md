# lungcancer-app
🫁 Lung Cancer Detection & Patient Assistant
An AI-powered diagnostic tool and interactive medical chatbot.

📝 Project Overview
This project aims to assist medical professionals and patients in the early detection of lung cancer using deep learning. The application provides a dual-layered approach: a high-precision CNN model for scan analysis and an Intelligent Chatbot for personalized health insights.

🚀 Key Features
AI Diagnostic Engine: Uses a MobileNetV2 architecture (Transfer Learning) to analyze CT scans and detect malignancy with high accuracy.

Damage Assessment: Calculates the percentage of lung damage and classifies severity levels (e.g., Healthy, Moderate, or Severe).

Contextual Chatbot: An integrated medical assistant that answers user queries based on their specific scan results and personal health profile.

User-Centric Interface: Developed with Streamlit to provide a seamless experience for uploading scans and viewing real-time results.

🛠️ Tech Stack
Deep Learning: Python, TensorFlow/Keras.

Web Framework: Streamlit.

Data Processing: NumPy, OpenCV.

Model Architecture: MobileNetV2 (Pre-trained on ImageNet).
